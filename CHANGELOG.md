## [1.0.0] – 2025-11-17

### Added
- Первая публичная сборка **Avenor Downloader** для Windows (инсталлер `.exe`).
- Вкладка **«Скачать»**:
  - Вставка ссылки из буфера.
  - Выбор качества (лучшее / 1080p / 720p / и т.д.).
  - Выбор папки сохранения.
  - История загрузок с прогресс-баром, статусами и кнопкой «Открыть в папке».
- Вкладка **«Сжать»**:
  - Выбор файла.
  - Задание целевого размера в МБ.
  - Сжатие с отображением прогресса и историей операций.
- Вкладка **«Конвертировать»**:
  - Выбор файла (видео / аудио / изображение).
  - Выбор целевого формата.
  - Выбор папки сохранения и история конвертаций.
- Вкладка **«Настройки»**:
  - Путь скачивания по умолчанию.
  - Включение/выключение звука «Готово» и «Ошибка».
  - Кнопка очистки истории.
  - Отображение текущей версии приложения.
- Система истории задач (download / compress / convert) с записью в `history.json`.
- Базовый обработчик автообновления (заглушка `app:checkUpdates`).

### Changed
- UI в фирменном стиле Avenor: тёмная тема, стеклянные карточки, неоновая зелёная кнопка «Вставить ссылку».
- Иконка приложения и инсталлера заменена на брендированный логотип Avenor.

### Known issues
- Автообновление ещё не реализовано — кнопка «Проверить обновления» отключена.
- Пока нет сборки для macOS (планируется в отдельном этапе).
